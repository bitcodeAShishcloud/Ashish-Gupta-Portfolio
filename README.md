 ## Getting Started

 Prerequisites: a modern web browser. Optionally install Node.js for local static serving.

 1. Clone the repository

 ```bash
 git clone https://github.com/bitcodeAShishcloud/cybersecurity-portfolio.git
 cd cybersecurity-portfolio
 ```

 2. Open locally

 - Open `index.html` in your browser, or run a simple static server:

 ```bash
 # Python 3
 python -m http.server 8000
 # or using Node (serve)
 npx serve .
 ```

 3. Customize

 - Update personal details in `index.html` (name, email, links).
 - Modify styles in `style.css`.
 - Update email settings in `app.js` if using EmailJS.
